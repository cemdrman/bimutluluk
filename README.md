Installing Instructions.

https://bimutluluk.herokuapp.com/api/records/

- Clone the project.
- Enter the project folder.
- Run "npm install".
- Run "npm start".

call `POST localhost:5000/api/records`

### Sample Request

```
{
	"startDate": "2016-01-26", 
	"endDate": "2018-02-02", 
	"minCount": 2980,
	"maxCount": 3000
}
```
